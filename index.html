使用指南

🚀 一键拉取

终端运行 npx serve -p 8080 → 用 localhost 打开本页面 → 点右上角 ⬇ 生成书签 → 在 HR 考勤页面点书签即可。

📋 手动粘贴

从 QHR 系统 Network 面板复制 alist 响应，粘贴到左侧输入框计算。

📊 导出报表

在日历视图中点击日期可切换类型（工作日/周末/节假日），确认无误后一键导出 Excel。
